This Landlord Simulation app is a web-based platform built with Flask that simulates a real-life interaction between potential tenants and landlords. The app allows users to browse various property listings, ask questions to the landlord (modeled by Google Gemini for natural conversation), and track their performance by asking relevant questions. I used SpaCy for natural language processing (NLP), specifically to calculate cosine similarity between the user's questions and a predefined set of "good questions.

NLP-Powered Chatbot

Python Flask React SpaCy
The Emotion Classification System is a machine learning project that uses computer vision and deep learning techniques to recognize human emotions based on facial and hand landmarks. I used OpenCV to capture real-time video and MediaPipe for extracting facial and hand landmarks from video frames. The landmarks are then processed and fed into a pre-trained TensorFlow model that classifies emotions such as happiness, sadness, and surprise.

Emotion Classification System

Python TensorFlow OpenCV
